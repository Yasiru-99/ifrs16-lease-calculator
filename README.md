# IFRS 16 Lease Calculator (Streamlit App)

A simple, interactive, and audit-friendly IFRS 16 lease accounting tool designed for auditors, accountants, and SMEs.

## 🚀 Overview

This application automates key IFRS 16 calculations, including lease liability measurement, right-of-use (ROU) asset depreciation, and journal entry generation. It is built using Python and Streamlit for ease of use and rapid deployment.

## ✨ Key Features

* Lease liability calculation using discounted cash flows
* Support for:

  * Ordinary annuity (end-of-period payments)
  * Annuity due (beginning-of-period payments)
* Date-based amortization schedule
* ROU asset depreciation (straight-line method)
* Automated journal entries:

  * Initial recognition
  * Periodic lease payments
  * Depreciation entries
* Export options:

  * CSV downloads
  * PDF report generation
* Simple and intuitive user interface

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* ReportLab

## 📦 Installation (Local Setup)

Clone the repository:

```
git clone https://github.com/Yasiru-99/ifrs16-lease-calculator
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the application:

```
streamlit run app.py
```
